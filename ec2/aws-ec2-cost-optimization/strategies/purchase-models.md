# Flexible Purchase Models

## Options
- **Savings Plans / Reserved Instances**: Commit for 1–3 years, save up to 72%.
- **Spot Instances**: Use for batch jobs, dev/test, save up to 90%.

## Best Practices
- Mix On-Demand + Reserved + Spot for balance.
- Use Auto Scaling groups with Spot for fault tolerance.

## Risks
- Reserved: Long commitments may backfire if workloads change.
- Spot: Instances can be interrupted anytime.
