# Auto Scaling + Load Balancing

## Overview
Scale EC2 capacity dynamically to match demand, avoiding idle costs.

## Steps
1. Configure Auto Scaling Groups.
2. Attach Elastic Load Balancer.
3. Define scaling policies (CPU, memory, request count).

## Benefits
- Pay only for what you use.
- Improves availability and resilience.

## Savings Potential
15–25% reduction in costs.
