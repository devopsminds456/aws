# Rightsizing EC2 Instances

## Overview
Rightsizing ensures workloads run on the most cost-effective instance type without over-provisioning.

## Tools
- AWS Compute Optimizer
- CloudWatch Metrics

## Steps
1. Analyze CPU, memory, and network utilization.
2. Identify underutilized instances.
3. Migrate to smaller instance families (e.g., t3.large → t3.medium).

## Savings Potential
20–30% reduction in monthly EC2 spend.

## Risks
- Downsizing too aggressively may cause latency spikes.
