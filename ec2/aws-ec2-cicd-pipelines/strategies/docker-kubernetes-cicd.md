# Docker & Kubernetes CI/CD

## Overview
Containerized workloads can be deployed to EC2 or EKS clusters.

## Steps
1. Build Docker images in CI pipeline.
2. Push to Amazon ECR.
3. Deploy to EC2/EKS via kubectl or Helm.

## Benefits
- Consistent environments
- Scalable deployments

## Risks
- Requires container orchestration expertise.
