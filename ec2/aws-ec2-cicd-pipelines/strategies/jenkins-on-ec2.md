# Jenkins on EC2

## Overview
Jenkins is a popular open-source CI/CD tool that can be hosted on EC2.

## Steps
1. Launch EC2 instance (Ubuntu preferred).
2. Install Jenkins via package manager.
3. Configure Jenkins with GitHub/Bitbucket integration.
4. Use Jenkins pipelines for build/test/deploy.

## Benefits
- Highly customizable
- Large plugin ecosystem

## Risks
- Requires manual scaling and maintenance.
