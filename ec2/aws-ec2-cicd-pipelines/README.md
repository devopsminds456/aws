# AWS EC2 CI/CD Pipelines

This repository documents **best practices** for implementing CI/CD pipelines on EC2.  
It is designed for **DevOps engineers, architects, and freelancers** who want to showcase practical AWS expertise.

## Key Strategies
- Jenkins on EC2
- GitHub Actions with EC2
- AWS CodePipeline integration
- Docker/Kubernetes CI/CD
- Blue-Green Deployments
- Monitoring & Rollbacks

Each strategy is explained with **examples, trade-offs, and client case studies**.
